## Human Help Network และบ้านจินใจ

สโมสรโรตารีพัทยามารีน่าให้การสนับสนุนมูลนิธิ Human Help Network และโครงการคุ้มครองเด็กในพัทยาอย่างแข็งขัน

### บริจาคให้ CPDC - พฤษภาคม 2022

ไฮดี้ เกลโม รองประธานสโมสรโรตารีพัทยามารีน่า ได้มอบเงินจำนวน **100,000 บาท** ให้แก่ศูนย์คุ้มครองและพัฒนาเด็ก (CPDC) ซึ่งบริหารจัดการโดย Human Help Network

### งาน Installation Night 2016

ในงาน Installation Night เมื่อเดือนมิถุนายน 2016 ประธาน โจ เคลมม์ และประธาน มิเชล โรช ได้มอบเงินจำนวน **18,000 บาท** ที่รวบรวมได้ให้แก่มูลนิธิ Human Help Network

### สถานสงเคราะห์เด็กบ้านจินใจ

สโมสรจัดส่งสิ่งของจำเป็นให้แก่สถานสงเคราะห์เด็กบ้านจินใจ หนองปรือ เป็นประจำ ซึ่งเป็นที่พักพิงของเด็กที่อยู่ในสถานการณ์เปราะบาง

### การแข่งขันเปตอง

เมื่อวันที่ 14 มีนาคม 2017 ได้จัดการแข่งขันเปตองประเภททีม 3 คน โดยมีเด็กจากสถานสงเคราะห์ร่วมทีมด้วย ตามด้วยงานเลี้ยงอาหารค่ำ เพื่อสนับสนุนมูลนิธิ
